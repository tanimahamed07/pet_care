Project name: Pet Care.

purpose:

Helping pet owners take care of their pets during cold weather.

live URL: https://per-care-kwu034u2c-tanim-ahameds-projects.vercel.app/


Key Features :
• Fully responsive design for mobile, tablet, and desktop
• Persistent Navbar and Footer across all pages
• Firebase Authentication: Email/Password login and Google login
• Protected Service Details Page: Only accessible after login
• Forgot Password feature with Gmail redirect
• User Profile Page showing name, email, and photo
• Update Profile functionality for user info
• Hero Slider showcasing pets in cozy winter outfits (using Swiper.js)
• Winter Care Tips and Expert Vets sections
• Dynamic Service Cards generated from JSON data
• Toast Notifications for success and error messages (using react-hot-toast)
• Animations for scroll, buttons, and cards (using AOS and Animate.css)
